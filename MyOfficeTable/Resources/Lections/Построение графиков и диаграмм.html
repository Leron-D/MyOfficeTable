Построение графиков и диаграмм

Диаграмма – это графическое представление данных отрезками или фигурами. В редакторе таблиц «МойОфис Таблица» доступны пять типов диаграмм:

Тип диаграммы

Визуальное представление

Гистограмма

Линейчатая диаграмма

Круговая диаграмма

Диаграмма с областями

График

К каждому типу диаграмм имеется дополнительные варианты - с накоплениями, а для некоторых графиков с возможностью показа маркеров и нормированных графиков.

Чтобы создать диаграмму или график, выполните следующие действия:

- выделите диапазон, по данным которого необходимо построить диаграмму или график,

- выберите в командном меню пункт «Вставка» и подпункт «Диаграмма»,

- в открывшемся подменю выберите тип диаграммы из возможных вариантов, представленных на рисунке 1 в зависимости от того, что требуется построить.

Рисунок 1 – Варианты выбора типа диаграммы

График/диаграмма сформируется автоматически по центру видимой области таблицы.

Данные, используемые для построения, подсвечиваются на листе цветами, которые соответствуют рядам данных диаграммы.

Чтобы перейти к настройкам диаграммы, выделите диаграмму и установите на нее указатель мыши. Щелчком правой клавиши мыши по диаграмме вызовите контекстное меню и выполните команду «Настройки диаграммы». На экране появится окно (см. рис. 2), которое содержит следующие инструменты:

- диапазон – можно изменить диапазон данных, формирующих диаграмму,

- название – можно указать название, отображаемое на диаграмме,

- задайте серии – можно указать способ размещения исходных данных диаграммы (по столбцам, по строкам),

- подписи – можно указать способ размещения подписей шкалы диаграммы (в первой строке и/или в первом столбце).

Рисунок 2 – Окно настройки диаграммы

Также, диапазон данных может быть добавлен позже.

В этом случае пользователь увидит окно с пустой диаграммой (см. рис. 3).

Рисунок 3 — Диаграмма без указания диапазона данных

Чтобы наполнить диаграмму данными, необходимо вызвать Настройки диаграммы (см. рис. 4).

Рисунок 4 — Окно Настройки диаграммы

Подписи доступны только на оси данных. Чтобы добавить подпись, при работе с диапазоном данных, необходимо включать в набор заголовки строки или столбца. Установка отметки в поле Подписи - Первая строка или Первый столбец добавит соответствующее наименование на изображение диаграммы (см. рис. 5).

Рисунок 5 — Окно Настройки диаграммы – Подписи

Чтобы изменить тип диаграммы достаточно нажать на соответствующую кнопку для каждого типа ( ) из списка доступных и диаграмма будет перестроена (см. рис. 6).
